How should inline task creation work?

You said no popup/modal — the form should be "flushed" into the task list. Here are two patterns:

A

Inline Row — Linear/Notion style

Click "+ New task" and an empty row appears inside the status group (e.g., inside the "Not Started" section). You type the title right there, then tab through fields (priority, assignee, due date) inline. Press Enter to save, Escape to cancel. No separate form — the new task IS a row.

B

Expandable Card — Asana/ClickUp style

Click "+ New task" and a card expands below the button with labelled fields (title, description, priority dropdown, assignee picker, dates). Fill in what you want, click "Create". More guided, more fields visible at once, but takes more vertical space.
